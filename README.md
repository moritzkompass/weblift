# weblift

Boutique für intelligente Websites.

## Struktur

- `index.html` — öffentliche Site mit Multistep-Anfrage-Quiz
- `admin/index.html` — Admin-Dashboard (Supabase-Auth)

## Stack

- **Frontend:** Static HTML/CSS/JS, gehostet auf GitHub Pages
- **Backend:** Supabase (Postgres + Auth + Storage)

## Domains

- Quiz: `weblift.at` (geplant)
- Admin: `admin.weblift.at` (geplant)
